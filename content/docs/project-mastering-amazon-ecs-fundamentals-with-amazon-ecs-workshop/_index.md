## 프로젝트 소개

이 프로젝트의 목표는 코드 분석과 랩 실습 내용을 문서화하는 과정에서 AWS 핵심 인프라와 Amazon ECS 기초 및 실무 스킬을 배우는 것입니다. 

이 프로젝트의 결과물은 각각의 포스트가 모여 하나의 document 형식을 가진 기술 문서가 될 것입니다.

## 프로젝트 학습 자료 소개

이 프로젝트에 사용될 학습 자료는 [Amazon ECS Workshop](https://catalog.workshops.aws/ecs-immersion-day/en-US)입니다. 그중 두 번째 러닝 패스인 "Deep Dive on Amazon ECS Features"가 메인 학습 자료가 될 것입니다. 

Amazon ECS Workshop은 2023년과 2024년에 열린 Amazon ECS Immersion Day 워크샵에서 사용된 학습 자료입니다. 이제는 더 이상 이 워크샵을 진행하지 않고, 대신 AWS ECS를 학습하고자 하는 사람들을 위해 외부에 공개된 것으로 보입니다.

이 학습 자료는 Amazon ECS에 대한 기초 지식과 실무에서 활용할 수 있는 스킬을 배우는 데 도움을 줍니다. 두 번째 러닝 패스인 "Deep Dive on Amazon ECS Features"는 ECS의 핵심적인 기능 하나하나를 랩 실습을 통해 배울 수 있는 컨텐츠입니다. 

## 프로젝트 진행 구조

이 프로젝트는 이 러닝 패스의 각 랩들을 진행하는 데 기반이 되는 인프라 코드를 먼저 분석하고, 그 후 각 랩 실습을 진행하면서 AWS ECS뿐만 아니라 ECS가 실행되는 AWS의 핵심 인프라도 같이 학습합니다. 실제 학습 자료에서는 인프라 코드를 직접 분석하는 과정은 따로 다루지 않습니다.
