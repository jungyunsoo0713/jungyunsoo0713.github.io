Amazon ECS Workshop의 두 번째 러닝 패스인 "Deep Dive On Amazon ECS Features"를 다룹니다.

각 랩은 크게 두 개의 파트로 나누어 살펴봅니다. 먼저 Infrastructure Code Analysis에서는 실습 환경을 구성하는 인프라 코드를 분석하고, Lab에서는 실제 실습 절차를 따라가며 리소스를 생성하고 동작을 확인합니다.

## Infrastructure Code Analysis

Amazon ECS Workshop이 제공하는 `EcsImmersionDay.json` CloudFormation 템플릿에서 해당 랩과 관련된 인프라 코드를 분석합니다.

## Lab

Amazon ECS Workshop에서 제공하는 실습 절차를 따라가며, 실제로 리소스를 생성하고 동작을 확인합니다.
